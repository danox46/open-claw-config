# Automation Debt Infographic - Image Brief

## Asset Overview
- **Type:** Single-image visual (infographic)
- **Title:** Automation Debt: 5 Warning Signs
- **Format:** PNG/JPEG
- **Recommended Size:** 1920x1080 (16:9 landscape)

## Visual Content

### Layout Structure
**Vertical flow with 5 equal sections, each containing:**
1. Large icon (64x64px or 96x96px)
2. Bold warning title
3. Brief explanatory text

### The 5 Warning Signs (Top to Bottom)

1. **Poor Exception Handling**
   - Icon: Broken chain / Alert symbol / X mark
   - Text: "No error recovery mechanisms in place"

2. **Data Quality Issues**
   - Icon: Corrupted file / Warning data symbol
   - Text: "Inconsistent or dirty input data"

3. **Brittle Edge Cases**
   - Icon: Fragile glass / Cracked icon
   - Text: "Breaks under unexpected scenarios"

4. **No Monitoring**
   - Icon: Empty dashboard / Eye closed
   - Text: "No visibility into performance"

5. **Over-Complicated Workflows**
   - Icon: Tangled knots / Complex maze
   - Text: "Too many unnecessary steps"

## Design Guidelines

### Style
- Clean, professional, tech-forward aesthetic
- Modern flat design or subtle gradient accents
- Consistent icon style (outline or filled)
- Brand color palette integration

### Typography
- Bold, clear headings (18-24pt)
- Concise supporting text (10-12pt)
- High contrast for readability

### Color Palette
- Primary: Brand primary color for icons
- Secondary: Neutral grays for text
- Accent: Warning/alert colors (amber/orange) for visual hierarchy

### Background
- Clean white or very light gray
- Optional subtle gradient overlay

### Icon Style
- Consistent line weight (2-3pt)
- Modern, minimalist approach
- Professional tech aesthetic
